Title: Scheduler Application

Purpose: 

	This application allows users to schedule appointments, manage customer data, and make reports.
	The business hours are 8:00 to 22:00 in Eastern Time Zone (New York). All dates and times
	are displayed in user's local timezone, however, all dates and times are converted into UTC Standard
	time zone format by the database driver manager, and saved into the database.

Directions for How to Run the Program:

	1- Download the assesment zipfile C195-Mehdi-Rahimi.zip and extract the Scheduler directory.
	2- Open the project in IntelliJ IDEA. 
	3- Select "Preferences..."
	4- Select "Path Variables" on the left.
	5- Set variable PATH_TO_FX to the directory of JavaFX library (e.g. /Library/javafx-sdk-19.0.2.1/lib).
	6- Select "Run" and then "Run 'Main'" on the menu bar.

Additional Report Description:
	
		User can see the total number of customers for a country selected on a combo box in the reports tab.
